# TacomaNarrows_DT

Tacoma Narrows Bridge — Digital Twin
A quarter-long research project exploring the implementation of digital twins, with a focus on the Tacoma Narrows Bridge (TNB).

Overview
This project builds a real-time digital twin of the Tacoma Narrows Bridge using the Godot game engine. The simulation models how individual bridge sections respond to incoming sensor data, visualizing structural behavior such as stress, deformation, and oscillation across discrete modular segments of the bridge.

Synthetic Data
Since live sensor data is not accessible, the simulation uses synthetic data modeled after real-world bridge sensor readings. All data streams are fully configurable, allowing different environmental and load scenarios to be tested. The bridge model responds to this data in real time.
